# Netflix OSS

This project demonstrates how to use important components from Netflix Open Source Stack.

The list of components includes Eureka, Hysterix, Kayron and Ribbon.

## How to run

| Step | Description | Command |
| :--: | :--- | :--- |
| 1 | Start Eureka | `./gradlew bootRun` |
| 2 | Start Pow service | `./gradlew run` |
